# License Wizard

A command line tool built with Go to easily create a license for your project

## Available Licenses 

```
* MIT
* BSD 2.0
* BSD 3.0
* APACHE 2.0
* AGPL 3.0
* GPL 3.0
* MPL 2.0
* CCZERO v1.0
```

## Example

```
licensewizard MIT
Author: <Your Name>
Project: <Your Project>
Output Directory: <Set Output Directory>
LICENSE generated and saved to <Output Directory>
```
